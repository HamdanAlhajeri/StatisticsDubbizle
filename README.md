# **StatisticsDubbizle

## Data Description
This data set contains information about rental properties in Abu Dhabi and details about the property.

## Column Names
1. Price
2. Nr Bedrooms 
3. Nr Bathrooms
4. Location
5. Size
6. Unnamed (Size Unit)
7. City
